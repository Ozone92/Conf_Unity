Pour faire fonctionner les diapos:
	-Mettre le jeu en play mode sur la scène "1_Basics"
	-Fleches gauche/droite pour avancer/reculer
	-P pour cacher ou ré-afficher la diapo 
	(si vous voulez afficher les exemples qu'il y a derrière)
	-R pour reset une diapo
